Glossary
========

.. glossary::

    experiment
        May refer to tokenizer training experiment or model training
        experiment. One usually train a tokenizer first and then train a model.

    experiment name
        Name of a particular :term:`experiment`.

    experiment path
        If :term:`experiment name` is ``my_exp``, then experiment path is
        ``exp/my_exp``.
        All :term:`experiment` related files will be put under directory ``exp``.

    Tokenizer
    tokenizer
        Tool for preprocessing text.
